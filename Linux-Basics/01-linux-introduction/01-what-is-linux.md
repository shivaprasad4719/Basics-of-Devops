# What is Linux?
<h4>Linux is an open-source operating system kernel used in servers, cloud platforms, containers, and DevOps environments.</h4>

---
# Why Linux for DevOps?
- Most servers run Linux
- Native support for automation & scripting
- Strong networking & process control
- Lightweight and secure
- Popular Linux Distributions
  * Ubuntu
  * Amazon Linux
  * CentOS / Rocky Linux
  * Red Hat Enterprise Linux (RHEL)
